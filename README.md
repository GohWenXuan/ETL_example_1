# HDB Dataset ETL
National datasets are readily available in https://data.gov.sg/. This script explores using API to access multiple datasets using Python and perform simple ETL


## Description
The primary goal is to ensure large datasets are downloaded, and ensure data quality, identify erroneous data and transform data according to business needs
